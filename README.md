# Excitation Machinery Parameter Prediction Method (EMPP)
### 本项目为一种基于非线性时变假设的果树分枝六自由度降阶模型仿真系统及振动采摘机械工作参数预测方法，通过大幅简化分枝自由度并结合数据驱动精确识别等效刚度参数与等效阻尼参数的方法，实现在田间针对不同果树的拓扑结构实时预测振动采摘机械的工作参数（包括启动时的初始工作参数以及陷入激励局部模态时的工作参数调整方向）。

---

* **采摘机械工作参数预测的实现方法与工作流：**
  * 参考目标果树，在模型中预设果树生长的拓扑结构；  
  * 参考目标果树，在模型中预设果实在分枝上的生长位置和数量；  
  * 参考试验结果，在模型中预设果柄与分枝的结合力；
  * 在模型中引入重力因素，在各分枝与果实的静力平衡位置的初始条件下开始后续计算；
  * 遍历所有可夹持的节点进行扫频实验，设计一个评价指标综合脱果率和振动响应大小等因素，判断激振机械的初始工作参数和夹持位置。

---

* **油茶果树的EMPP最终实现：**
  * 可以根据输入的拓扑结构自定义模型自由度。——可仿真任意数量任意层级果树；
  * 父级分枝和子级分枝的连接节点可以调节。——可适应二级分枝生长在一级分枝不同位置；
  * 可以计算分枝各个节点的二维运动轨迹；
  * 可定义果实生长位置。——符合实际果树上的果实生长位置不同的情况；
  * 模型考虑重力影响。——全部果实与分枝均在重力条件下的初始平衡点进行后续计算；
  * 模型包含前处理数据识别，通过优化理论计算出的频响函数 (FRF) 与实验测量的 FRF 最小误差值分配各个节点的等效参数。——智能识别哪里更硬，哪里更软，而不是直接猜；
  * 模型对每个分枝的每个节点都独立进行非线性检测，只有被判定为具有显著非线性的节点，才会被分配非线性参数（谐波平衡法）。——平衡计算时间和计算精度；
  
---

### 实验方案：
1.**振动衰减实验**：在各个分枝的根部、中段、末端固定三轴加速度传感器，使用力锤分别沿着重力方向和分枝径向方向敲击，敲击顺序为根部→中段→末端，记录各位置加速度衰减曲线。  
2.**定频采摘试验**：利用手持式摇果机定频激励某一级分枝，测量各个分枝三个部分的加速度曲线。
